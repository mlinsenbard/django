Spellimobulator works by calucalating the DPS of all damaging champion spells, and returning the top 5 results. 100+ of these spells have DPS values calculated by hand since they require more attention than a standard 'nuke' spell.

All spells are calculated with the following criteria:
1. Damage is only dealt to one, 'champion' target. (So no, Nunu's Consume is not considered a Damaging spell to Champions) 2. No mana, enery, or other resource is taken into consideration. Spellimobulator assumes these resources are infinite. 3. No previously active spells, passives, or other abilites are considered when caluclating DPS. Passives are also not considered in these calculations at all. 4. These calculations are in no way accurate, as no resistances are taken into consideration. 5. Spells that do percent health damage will not have that damage considered as the target's health is not a given variable.

If you would like to see more on the Spellimobulator, visit the github repository for it here. If you want to see how I hooked the python code into my website, visit the github repo for my website. This web version will be more up to date than the standalone version, so be sure to check for updates.
